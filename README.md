# Weather Dashboard

Public Streamlit dashboard for weather station data stored in NetCDF format.

## Features
- Temperature
- Humidity
- Wind speed
- Solar radiation
- Rain accumulation
- Lightning strike distance
- Secure HTTPS (Streamlit Cloud)

## Run locally
```bash
streamlit run weather_dashboard.py
